# 问题反馈与社区

## 问题反馈

遇到 Bug 或有功能建议，欢迎通过以下渠道反馈：

- [GitHub Issues](https://github.com/ys-ll/uniterm/issues)

提交 Issue 时请尽量包含：
- 操作系统和版本
- uniTerm 版本号
- 问题描述和复现步骤
- 相关截图或日志
