# 开始页

开始页是 uniTerm 启动后默认显示的页面，提供快速访问入口。关闭所有标签页后也会自动打开。

![开始页](/imgs/start_tab_light.webp)

## 搜索与快速连接

顶部搜索框支持按名称、主机、协议类型搜索已保存的连接。同时也是一个快速连接入口——输入主机地址后，搜索结果末尾会出现「快速连接」卡片，双击即可按该地址新建连接。支持解析 `ssh user@host:22` 等协议前缀格式。

## 连接卡片

连接以卡片网格形式展示，每张卡片显示协议图标、连接名称和摘要信息（如 `SSH user@host:22`）。

**鼠标操作：**

- **单击** — 选中卡片。`Ctrl + 单击` 多选，`Shift + 单击` 范围选择
- **双击** — 打开连接
- **右键** — 弹出上下文菜单，支持连接、编辑、切换协议（SSH 可切换 SFTP/监控）、修改分组、复制连接、删除
- **批量操作** — 多选后右键可批量删除或批量修改分组

**键盘操作：**

- `Tab` — 在搜索框和卡片网格之间切换焦点
- `↑` `↓` `←` `→` — 在卡片间移动焦点
- `Enter` — 打开当前焦点的连接（多选时打开全部选中连接）
- `Esc` / `Backspace` — 从分组视图返回首页

## 快速入口

搜索框下方三个操作按钮：

- **新建连接** — 弹出新建连接对话框
- **本地终端** — 下拉选择已安装的 Shell（PowerShell / CMD / Git Bash / WSL / bash / zsh 等）
- **连接串口** — 快速打开串口连接

## 最近连接

显示最近打开过的连接记录（最多 20 条），按使用时间排序，点击或双击可快速重连。

## 连接分组

以卡片形式展示所有连接分组，显示分组名称和连接数量。未分组连接归入「未分组」。

- **进入分组** — 单击或双击分组卡片，进入分组详情视图，仅显示该分组内的连接
- **面包屑导航** — 分组视图顶部显示 `首页 / 分组名`，点击首页返回
- **新建分组** — 点击分组旁的 `+` 按钮创建
- **右键** — 重命名或删除分组。删除时可选保留或移除组内连接
