# 常见问题与排错

## 连接问题

### SSH 连接失败，提示 "Connection refused"

- 确认目标服务器 SSH 服务是否在运行
- 检查 IP 地址和端口是否正确
- 检查防火墙是否允许该端口

### SSH 密钥认证失败

- 确认密钥文件路径和权限正确
- 确认服务器 `~/.ssh/authorized_keys` 中包含对应公钥
- 尝试使用密码认证验证服务器可达性

### RDP 白屏或卡住

- 检查目标主机是否已启用远程桌面
- 尝试降低分辨率或颜色深度
- Windows Home 版本不支持 RDP 服务端

### 串口连接无反应

- 确认串口号正确（Windows 下为 COMx，Linux 下为 /dev/ttyUSBx）
- 检查波特率等参数是否与设备一致
- 尝试开启"本地回显"以确认输入是否发出

## 功能问题

### AI 助理不响应

- 确认 API 地址和密钥配置正确
- 检查网络是否可访问 API 端点
- 查看 AI 模型配置页面的连接测试结果

### 云同步失败

- 确认 Git 仓库地址和访问令牌正确
- 确认仓库为私有仓库
- 检查网络是否可访问 Git 服务
