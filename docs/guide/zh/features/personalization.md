# 个性化

uniTerm 提供丰富的个性化选项，让终端符合您的使用习惯和审美偏好。


## 应用配置

### 界面主题

3 种 UI 主题：

| 主题 | 说明 |
|------|------|
| Dark | 经典暗色，护眼舒适 |
| Deep Blue | 深蓝科技感 |
| Light | 明亮清新 |

### 语言

支持 9 种界面语言，切换后无需重启即可生效：

- 简体中文
- 繁體中文
- English
- 日本語
- 한국어
- Deutsch
- Español
- Français
- Русский


## 终端配置

在「终端」标签页中可调整终端的外观和行为。

![终端设置](/imgs/terminal_settings_light.webp)

### 配色方案

内置 **27** 款终端配色方案，涵盖 Solarized、Dracula、Monokai、Nord、Gruvbox、Catppuccin、Tokyo Night 等流行主题。同时支持**自定义配色**，可单独调整前景色、背景色及 16 种 ANSI 颜色，自由搭配出专属配色方案。支持导入/导出 `.itermcolors` 格式的配色文件。

### 字体

- **字体家族** — 下拉列表自动获取系统已安装字体，选择后实时预览
- **字号** — 范围 8–32，默认 14

### 回滚行数

设置终端可保留的最大输出行数，范围 100–50,000，默认 2,500。超出后早期输出将被丢弃。

### 选中行为

- **无操作** — 选中文本后不做任何处理
- **复制到剪贴板** — 选中文本时自动复制

### 右键行为

- **显示右键菜单** — 右键弹出上下文菜单
- **粘贴剪贴板内容** — 右键直接粘贴

### 中键行为

- **无操作** — 中键点击无反应
- **粘贴剪贴板内容** — 中键单击直接粘贴（默认）

### 智能补全

开启后在终端输入时，根据命令历史和 AI 提供实时补全建议。

### 文本高亮

开启后自动高亮终端输出中的时间戳、IP 地址、关键字、字符串等内容。


## 快捷键

uniTerm 支持自定义快捷键。

![快捷键设置](/imgs/shortcuts_light.webp)

在设置 → 快捷键中，搜索或浏览所有可绑定操作，点击后按下目标组合键即可完成绑定。支持单键和组合键（含 `Ctrl` / `Alt` / `Shift` / `Meta`）。
