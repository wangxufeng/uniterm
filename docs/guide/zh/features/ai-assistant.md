# AI 助理

uniTerm 内置自主 AI Agent，可在终端中独立规划并执行多轮 Shell 命令。

![AI 助理](/imgs/ai_assistant_light.webp)

## 核心能力

### 自主多轮执行

AI Agent 会：

1. **理解意图** — 分析您输入的自然语言需求
2. **制定计划** — 拆解为可执行的 Shell 命令步骤
3. **执行观察** — 在终端中运行命令，观察输出结果
4. **迭代调整** — 根据结果修正计划，继续执行直到完成任务

### LLM 提供商配置

支持 Anthropic 和 OpenAI 兼容 API，可添加多个模型并在使用中随时切换。

![AI 模型设置](/imgs/ai_model_light.webp)

**模型配置字段：**

| 字段 | 说明 |
|------|------|
| 名称 | 自定义显示名称，用于在模型列表中区分不同配置 |
| 协议 | 选择 **Anthropic** 或 **OpenAI** 协议，决定请求格式 |
| Base URL | API 端点地址，OpenAI 默认为 `https://api.openai.com/v1`，Anthropic 默认为 `https://api.anthropic.com` |
| User-Agent | 可选，自定义请求头中的 User-Agent 标识，提供 uniTerm、Claude Code、Cursor 等预设，也支持手动输入 |
| API Key | API 认证密钥，以密码形式存储 |
| 模型 | 模型名称（如 `gpt-4o`、`claude-sonnet-4-20250514`），可通过「获取模型列表」按钮从 API 自动拉取可选模型 |
| 测试连接 | 填写完 API Key 和模型后，点击「测试连接」验证配置是否正确 |

**多模型管理：**

- 支持添加多个模型配置，每个独立设置协议、地址和密钥
- 在 AI 侧边栏顶部下拉菜单中切换当前使用的模型
- 设置中可编辑或删除已有模型配置

### 执行模式

| 模式 | 说明 |
|------|------|
| 跳过 | 所有命令直接执行，无需确认 |
| 仅危险命令 | 潜在危险命令（rm、sudo 等）需确认 |
| 危险 + 写入 | 危险命令和文件写入操作需确认 |
| 全部确认 | 每一条命令都需要手动确认 |

### 终端集成

AI 助理与终端深度集成，命令直接在终端标签页中执行，无需手动复制粘贴。

**执行方式：**

1. 在 AI 对话框中输入自然语言需求（如"帮我检查磁盘使用情况"）
2. AI 分析需求后生成执行计划
3. 根据当前执行模式，可能需要您确认每条命令
4. 命令在目标终端中逐条执行，输出实时显示
5. AI 观察输出结果，自动调整后续步骤

**选择目标终端：**

- **跟随活动标签页**（默认）— AI 命令在当前激活的终端标签页中执行。切换标签页后，后续命令自动切换到新标签页
- **固定到指定标签页** — 点击 AI 面板顶部的固定按钮，选择某个已打开的终端标签页。之后所有 AI 命令都在该标签页中执行，不受切换影响

**分屏协作：**

拖拽 AI 对话面板到右侧区域，与终端形成左右分栏。一边查看 AI 的分析和计划，一边观察终端中的执行结果，互不遮挡。
