# 智能提示

在终端输入时，uniTerm 提供实时补全建议，帮助您更快地输入命令。

![智能提示](/imgs/auto_complete_light.webp)

## 智能补全

在终端中键入命令时，输入框下方自动弹出补全建议，按 `↑` `↓` 选择、`Enter` 接受、`Esc` 关闭。补全来源包括：

- **历史匹配** — 从历史记录中按前缀和模糊匹配，匹配字符高亮显示
- **快捷命令** — 匹配已保存的快捷命令名称和命令内容
- **AI 建议** — AI 根据当前输入上下文生成命令建议，结果到达后自动选中

在设置中可全局开启或关闭智能补全。

## 快捷命令

将常用命令保存为快捷命令，可在左侧面板中随时调用。

### 管理快捷命令

- **新建命令** — 填写命令名称（可选）和命令内容，选择所属分组
- **新建分组** — 创建分组用于整理快捷命令
- **拖拽排序** — 命令可在分组之间拖拽移动
- **右键菜单** — 编辑命令、删除命令，或者重命名/删除分组

### 执行方式

每条快捷命令支持三种操作：

- **运行** — 发送命令 + 回车，直接执行
- **粘贴** — 仅粘贴命令文本到终端，不自动执行
- **复制** — 复制命令到剪贴板

## 终端历史

uniTerm 自动记录终端中输入的命令，去重后保存在本地。

- **历史面板** — 按 `Ctrl + Shift + H` 打开，支持搜索和多选
- **运行 / 粘贴 / 复制** — 与快捷命令相同的三种执行方式
- **保存为快捷命令** — 选中历史记录，一键转为快捷命令
- **删除** — 移除不需要的历史记录
