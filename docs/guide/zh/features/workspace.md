# 标签和工作区

uniTerm 通过标签页管理多个连接，通过工作区将多个终端分栏并排展示。

![工作区](/imgs/workspace_light.webp)

## 标签页

### 标签页操作

- **新建标签页** — 双击左侧连接列表中的连接，或右键连接选择连接方式
- **切换标签页** — 点击顶部标签页切换
- **关闭标签页** — 点击标签页右侧 × 按钮，或右键关闭

### 标签页拖拽

- **拖拽排序** — 拖动标签页可调整排列顺序
- **拖入分栏** — 将终端类型的标签页拖入内容区域可创建分栏（其他类型标签页不支持合并到工作区）
- **拖出合并** — 将分栏中的标签页拖回标签栏可取消分栏

### 标签页右键菜单

右键标签页弹出菜单，不同标签类型显示不同项：

**通用菜单（所有标签页）：**
- 重命名 — 修改标签页显示名称
- 关闭 — 关闭标签页

**终端类菜单（SSH / Local）：**
- 复制会话 — 复制当前连接，打开新的标签页
- 文本搜索 — 在终端输出中搜索关键字
- 导出文本 — 将终端输出导出为文本文件

**SSH 专属菜单：**
- 打开 SFTP — 在当前连接上打开 SFTP 文件浏览器
- 上传文件 — 向终端发送 `rz -be` 命令
- 打开服务器监控 — 在当前连接上打开监控面板

## 工作区

### 创建工作区

- **拖拽标签页** — 将标签页拖入内容区域，拖到上/下边缘创建水平分栏，拖到左/右边缘创建垂直分栏
- **调整大小** — 拖动分栏之间的分隔线调整各面板比例
- **关闭分栏** — 将标签页拖回标签栏即可取消分栏

### 面板菜单

每个终端面板标题栏右侧提供操作按钮：

- **广播** — 开启后当前面板输入同步到工作区内所有终端
- **AI 锁定** — 将 AI 助理固定到该面板，锁定后面板标题栏高亮
- **更多** — 下拉菜单（与标签页右键菜单类似）：

  终端类菜单：
  - 复制会话
  - 文本搜索
  - 导出文本

  SSH 专属菜单：
  - 打开 SFTP
  - 上传文件
  - 打开服务器监控
- **关闭** — 关闭当前面板

### 广播输入

开启广播输入后，在当前工作区任一终端中输入的内容会同时发送到该工作区的所有终端。适用于同时在多台服务器上执行相同命令的场景。
