# 远程终端

uniTerm 支持 **SSH**、**Telnet** 和 **Mosh** 三种远程终端协议。

## SSH

SSH（Secure Shell）是最常用的远程服务器管理协议，提供加密的安全连接。

### 连接参数

| 参数 | 说明 |
|------|------|
| 主机 | 服务器 IP 或域名 |
| 端口 | 默认 22 |
| 用户名 | 登录用户名 |
| 认证方式 | 密码或密钥 |
| 密码 | 密码认证时填写 |
| 密钥路径 | 密钥认证时选择私钥文件 |
| 字符编码 | 终端字符集，默认 UTF-8，可选 GBK、GB2312、Big5、Shift-JIS 等 |
| SSH 隧道 | 选择已有的 SSH 连接作为跳板机，通过其转发流量到目标主机 |
| 启动脚本 | 连接建立后自动执行的命令，每行一条，顺序执行 |

### 认证方式

- **密码认证** — 输入密码直接登录。如未保存密码，连接时会弹出终端提示输入
- **密钥认证** — 选择 SSH 私钥文件（如 `~/.ssh/id_rsa`）进行无密码登录
- **键盘交互认证** — 始终作为备用方式，当密码或密钥认证失败时自动启用交互式认证

### SSH 隧道

可通过 SSH 隧道将其他连接的流量经跳板机转发，实现内网穿透。选择已有的 SSH 连接作为跳板机后，目标主机的流量会通过该 SSH 连接加密转发。支持为隧道指定不同的用户名和密码。

### Post-Login Expect

Expect/Send 模式支持交互式登录自动化，按顺序定义匹配规则和发送内容：

- **expect** — 等待终端输出匹配该文本（支持正则，不区分大小写）
- **send** — 匹配成功后发送的文本，支持 `${password}`、`${user}`、`${host}` 变量
- **enter** — 发送后是否追加回车（默认是）
- **timeout** — 单步超时时间（默认 10 秒，最大 120 秒）

### Zmodem 文件传输

在 SSH 终端中，可直接使用 `rz`（接收文件）和 `sz`（发送文件）命令传输文件，无需打开文件浏览器。也支持从桌面拖拽文件到终端窗口直接上传。

### SSH 终端标签页操作

标签页支持拖拽排序、拖入内容区分栏。右键标签页弹出菜单：

| 菜单项 | 说明 |
|--------|------|
| 复制会话 | 复制当前 SSH 连接，打开新的标签页 |
| 打开 SFTP | 在当前连接上打开 SFTP 文件浏览器 |
| 上传文件 | 向终端发送 `rz -be` 命令，接收文件到远程 |
| 打开服务器监控 | 在当前连接上打开服务器监控面板 |
| 文本搜索 | 在终端输出中搜索关键字 |
| 导出文本 | 将当前终端输出导出为文本文件 |

## Telnet

Telnet 是一种简单的明文远程终端协议，常用于连接嵌入式设备、网络设备和遗留系统。

### 连接参数

| 参数 | 说明 |
|------|------|
| 主机 | 设备 IP 或域名 |
| 端口 | 默认 23 |
| 用户名 | 登录用户名（可选，连接后自动发送） |
| 密码 | 登录密码（可选，用户名发送后自动发送） |
| 启动脚本 | 连接建立后自动执行的命令 |

> ⚠️ Telnet 不加密传输数据，请仅在可信网络中使用。

## Mosh

Mosh（Mobile Shell）专为高延迟和间歇性网络设计，基于 UDP 传输，在网络切换（如 Wi-Fi 切 4G）时连接不中断。

### 连接参数

与 SSH 相同，uniTerm 先通过 SSH 连接到服务器，再自动启动 `mosh-server` 建立 UDP 会话。

| 参数 | 说明 |
|------|------|
| 主机 | 服务器 IP 或域名 |
| 端口 | SSH 端口，默认 22 |
| 用户名 | 登录用户名 |
| 认证方式 | 密码或密钥 |

> 注意：使用 Mosh 前，服务器需安装 `mosh-server`。Mosh 不支持 SSH 隧道和 Expect/Send 模式。
