# 文件传输

uniTerm 内置双栏文件浏览器，支持多种文件传输协议，提供统一的文件管理界面。

## 支持的协议

### SFTP

基于 SSH 的安全文件传输协议。连接参数与 SSH 共用——主机、端口（默认 22）、用户名、密码或密钥认证。

### FTP / FTPS

传统 FTP 协议及加密的 FTPS。

| 参数 | 说明 |
|------|------|
| 主机 | FTP 服务器地址 |
| 端口 | 默认 21 |
| 用户名 / 密码 | 登录凭据 |
| 加密 | 可选 FTPS（显式/隐式 TLS） |

### SMB

Windows 文件共享协议（Samba），用于连接局域网共享文件夹或 NAS。

| 参数 | 说明 |
|------|------|
| 主机 | SMB 服务器地址 |
| 端口 | 默认 445 |
| 共享路径 | 共享目录路径 |
| 用户名 / 密码 | 登录凭据 |
| 域 | 可选，Windows 域环境使用 |

### WebDAV

基于 HTTP 的文件管理协议，常用于 NextCloud、ownCloud 等云存储服务。

| 参数 | 说明 |
|------|------|
| URL | WebDAV 服务地址 |
| 用户名 / 密码 | 登录凭据 |

### S3

兼容 Amazon S3 API 的对象存储，也支持 MinIO、阿里云 OSS 等兼容服务。

| 参数 | 说明 |
|------|------|
| Endpoint | 服务端点地址 |
| Access Key | 访问密钥 ID |
| Secret Key | 访问密钥 |
| Bucket | 存储桶名称 |
| Region | 区域（可选） |


## 文件浏览器

连接建立后，所有协议共用同一套文件浏览器界面——双栏布局，左侧为本地文件系统，右侧为远程目标。

![文件传输](/imgs/sftp_light.webp)

### 双栏布局

左侧为本地文件系统，右侧为远程目标，中间分隔线可拖动调整左右比例。顶部各有独立的路径栏和工具栏。

### 文件操作

- **上传** — 选中本地文件拖拽到远程侧、从桌面/资源管理器拖拽文件到窗口中，或点击工具栏上传按钮
- **下载** — 选中远程文件拖拽到本地侧、拖拽到桌面/资源管理器，或点击工具栏下载按钮
- **多选** — 按住 `Ctrl` 逐个多选，`Shift` 范围选择
- **新建文件夹** — 在当前位置创建新目录
- **删除** — 选中文件/文件夹后删除（支持批量）
- **重命名** — 右键或单击已选中的文件名进行重命名
- **复制 / 移动** — 在同一侧复制或移动文件到其他目录

### 右键菜单

在文件列表中右键弹出上下文菜单，提供新建文件夹、删除、重命名、刷新、复制路径等常用操作。

### 路径导航

- **路径栏** — 显示当前目录的完整路径，点击任意一级目录可快速跳转
- **手动输入** — 直接在路径栏输入路径后回车跳转
- **路径收藏** — 收藏常用路径，点击星标图标添加，收藏列表一键跳转
- **返回上级** — 点击 `..` 或工具栏按钮返回上一级目录

### 文件列表

- **列表视图** — 以表格形式显示文件名、大小、修改时间、权限等信息
- **排序** — 点击列头按名称、大小、修改时间等字段升降序排列
- **隐藏文件** — 可切换是否显示以 `.` 开头的隐藏文件
- **刷新** — 手动刷新当前目录内容

### 文本编辑器

双击文本文件可在内置编辑器中打开，支持字符编码切换、换行符类型切换（LF/CRLF）、保存到远程。

### 传输管理

- **传输进度** — 底部状态栏显示当前传输进度
- **传输队列** — 多个传输任务排队执行，可暂停或取消单个任务
