# 数据库

uniTerm 内置数据库客户端，支持主流关系型数据库和 Redis。

## 关系型数据库

![数据库](/imgs/database_light.png)

### 支持的数据库

| 数据库 | 默认端口 |
|--------|----------|
| MySQL | 3306 |
| PostgreSQL | 5432 |
| Oracle | 1521 |
| SQL Server | 1433 |
| rqlite | 4001 |

### 连接参数

| 参数 | 说明 |
|------|------|
| 主机 | 数据库服务器地址 |
| 端口 | 各数据库默认端口自动填充 |
| 用户名 | 数据库登录用户 |
| 密码 | 数据库登录密码 |
| 数据库名 | 连接后默认使用的数据库（rqlite 不需要） |
| SSH 隧道 | 开启后可通过 SSH 跳板机加密连接内网数据库，无需暴露数据库端口 |

### 数据库导航

左侧树形面板展示数据库层次结构，连接后可浏览库、表、视图三级对象。

- **展开/折叠** — 点击库名左侧箭头展开表列表，表以网格图标显示，视图以眼睛图标区分
- **搜索** — 顶部搜索框输入关键字，实时过滤表名和视图名，搜索时自动展开所有库
- **双击打开** — 双击库名进入该库的查询页面，双击表名或视图名进入数据浏览页面

右键菜单提供以下操作：

- **新建** — 新建数据库、新建表
- **查询** — 打开 SQL 查询或数据浏览
- **管理** — 查看表结构、清空表、删除表/视图/库
- **刷新** — 重新加载表列表

### 数据查询

- **SQL 查询** — 输入 SQL 后点击执行或按 `Ctrl+Enter` 运行，结果以表格展示，NULL 值以斜体显示。双击表名默认查询该表前 100 条数据
- **新增行** — 自动检测列类型、默认值和自增列，生成空白插入表单
- **编辑行** — 点击铅笔图标打开编辑表单，支持所有列修改和 NULL 切换（需查询结果包含主键列）
- **删除行** — 点击删除图标，确认后删除（需查询结果包含主键列）

### 表和视图列表

双击库名进入，以表格形式展示该库下所有表和视图，支持搜索和排序。每行可执行清空表、删除表/视图，危险操作需输入名称确认。

### 表结构浏览

双击表名进入，左侧树面板展示列信息和索引结构，包括列名、数据类型、是否可空、默认值、主键、自增等详情。

- **编辑列** — 修改列类型、默认值、注释、排序规则
- **新增列** — 添加新列，支持类型自动补全提示
- **删除列** — 确认后删除
- **索引管理** — 新增、删除索引

## Redis

Redis 提供键值数据浏览和管理功能。

![Redis](/imgs/redis_light.png)

### 连接参数

| 参数 | 说明 |
|------|------|
| 主机 | Redis 服务器地址 |
| 端口 | 默认 6379 |
| 用户名 | Redis ACL 用户名（可选，Redis 6+） |
| 密码 | Redis 认证密码（可选） |
| SSH 隧道 | 开启后可通过 SSH 跳板机加密连接内网 Redis |

### 键浏览

左侧面板展示所有 Key 的列表，显示 Key 名称和数据类型，支持按名称搜索过滤。选中 Key 后在右侧查看值内容，不同类型以对应格式展示。

右键菜单提供以下操作：

- **新建** — 新建 Key
- **编辑** — 修改值、重命名
- **管理** — 删除 Key（需确认）、设置或移除过期时间
- **刷新** — 重新加载键列表

### 键操作

- **新建 Key** — 选择数据类型后创建新键，输入 Key 名和值
- **查看/编辑值** — 选中 Key 后查看值内容，点击编辑进入修改模式并保存
- **重命名 Key** — 修改 Key 名称
- **删除 Key** — 选中 Key 后删除，需确认

### TTL 管理

- **设置过期** — 新建或编辑 Key 时设置过期时间
- **移除过期** — 取消过期时间，使 Key 永久保存
